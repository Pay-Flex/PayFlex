Processus centre

Les clients smartphone peuvent payer via FedaPay (sandbox) : validation automatique à la confirmation du paiement. Sinon déclaration classique → agent ou validation auto après 24 h sans réponse.

Les espèces saisies par l'agent sur mobile sont confirmées immédiatement. Les espèces saisies au centre restent à confirmer manuellement (Actions : Confirmer / Refuser).

Le client reçoit une notification dans l'app dès qu'une cotisation est confirmée ou refusée. Utilisez le bouton Espèces en attente au-dessus du tableau pour le rapprochement caisse.
