Pourquoi ces montants apparaissent ici

Quand une cotisation validée dépasse ce qu'il reste à payer sur le produit visé, PayFlex répartit automatiquement l'excédent sur les autres produits actifs du client. Si aucun produit actif n'est disponible (tous complets), le reliquat est conservé ici — l'argent n'est jamais perdu, mais doit être régularisé manuellement : réaffectation à un nouveau produit du client, ou remboursement / traitement hors système.
